  - **Fork** the Repository
  - Open your **Terminal / Cmd**
  
 <br>
 
   ```
   $ git clone https://github.com/<your username>/SBC.git  
   ```   

   ```
   $ cd SBC
   ```

   ```
   $ pip install virtualenv
   $ python -m venv <name of environment>
   $ source <name>/bin/activate
   $ pip install -r requirements.txt
   ```

   ```
   $ python main.py
   
   OR
   
   $ bash run.sh 
   ```
